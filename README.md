# DessertOrderApp
Mocking Data를 활용한 내 주변의 카페를 찾고, 주문할 수 있는 앱

## 구현 기능
1. 내 위치를 기반으로 주위 카페 리스트 보여주기 (원하는 위치로 조정 가능)
2. 찜 기능
3. 주문 기능
4. 로그인 기능
